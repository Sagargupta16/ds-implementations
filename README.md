# 📚 Data Structures & Algorithms Implementations

A comprehensive collection of fundamental data structures and algorithms implemented in C++. This repository serves as a learning resource and reference for computer science students and developers.

## 🔍 Overview

This repository contains well-documented implementations of core data structures and algorithms, organized into clear sections for easy navigation and understanding. Each implementation focuses on clarity, efficiency, and educational value.

## 📂 Repository Structure

```text
DS-Implementations/
├── 01_Recursion/          # Recursive algorithms and patterns
├── 02_Arrays/             # Array operations and manipulations
├── 03_ArrayADT/           # Array Abstract Data Type
├── 05_Matrices/           # Matrix operations and special matrices
├── 06_SparseMatrix/       # Sparse matrix implementations
├── 07_LinkedList/         # Singly, doubly, and circular linked lists
├── 08_Stack/              # Stack implementations and applications
├── 09_Queue/              # Queue variants and operations
├── 10_Tree/               # Binary trees and tree traversals
├── 11_BST/                # Binary Search Trees
├── 12_AVLT/               # AVL Trees (Self-balancing BST)
├── 13_Search Tree/        # Advanced search tree variants
├── 14_Heap/               # Heap data structure and heap sort
├── 15_SortingTechs/       # Various sorting algorithms
├── 16_HashingTechs.cpp/   # Hash tables and collision resolution
└── 17_Graphs/             # Graph representations and algorithms
```

## 🚀 Key Features

- **Educational Focus**: Clear, readable code with educational comments
- **Comprehensive Coverage**: From basic arrays to advanced graph algorithms
- **Multiple Implementations**: Different approaches for the same concepts
- **Real-world Applications**: Practical examples and use cases
- **Performance Analysis**: Time and space complexity discussions

## 📖 Topics Covered

### 🔄 Recursion

- Factorial, Fibonacci sequences
- Tower of Hanoi
- Tree recursion patterns
- Tail and head recursion
- Taylor series implementations

### 📊 Linear Data Structures

- **Arrays**: Operations, searching, sorting
- **Linked Lists**: Singly, doubly, circular variants
- **Stacks**: Array and linked list implementations
- **Queues**: Simple, circular, priority queues

### 🌳 Non-Linear Data Structures

- **Trees**: Binary trees, BST, AVL trees
- **Heaps**: Min/max heaps, heap sort
- **Graphs**: Adjacency matrix/list, traversals, MST

### 🔍 Algorithms

- **Sorting**: Bubble, selection, insertion, merge, quick, heap sort
- **Searching**: Linear, binary search variants
- **Graph Algorithms**: BFS, DFS, Dijkstra, Prim, Kruskal
- **Hashing**: Chaining, linear probing, quadratic probing

## 🛠️ Getting Started

### Prerequisites

- C++ compiler (GCC, Clang, or MSVC)
- Basic understanding of C++ programming

### Compilation

```bash
# For individual files
g++ -o program_name file_name.cpp

# Example
g++ -o factorial 01_Recursion/Factorial.cpp
./factorial
```

### Usage

1. Navigate to the desired topic folder
2. Choose the implementation you want to study
3. Compile and run the program
4. Study the code and comments for understanding

## 🎯 Learning Path

**Beginner**: Start with Arrays → Recursion → Linked Lists
**Intermediate**: Move to Stacks → Queues → Trees → Sorting
**Advanced**: Progress to Graphs → Hashing → Advanced Trees
